 # Bagh-Chal Game

## Description
This project is a web-based implementation of the traditional Bagh-Chal strategy board game. 
Players can choose to play against AI or another player.

## Features
- Login page
- Play vs AI
- Play vs Player
- Interactive 5x5 board
- Turn-based gameplay

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project files
2. Open the folder
3. Double click `index.html`
4. The game will open in your browser
